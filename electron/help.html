Getting Started

Using the Command Palette

The Command Palette is the fastest way to navigate and perform actions. It allows you to do almost anything without taking your hands off the keyboard.

How to Use It:

  1. Press Ctrl+K (or Cmd+K on Mac) to open the palette.
  2. Start typing what you want to do. For example:
    • Type Create new note to make a new note.
    • Type Go to settings to open the settings page.
    • Type the name of a note or snippet (e.g., My API notes) to find and open it.
  3. Use the arrow keys (↑ and ↓) to select a result and press Enter to execute the action.

Projects

Managing a Project

The Projects section is a dedicated workspace to organize all aspects of a single initiative, from high-level goals to specific code snippets.

Step-by-Step Instructions:

  1. Create a Project: Navigate to the Projects page from the sidebar and click the New button. A new project will be created and selected for you.
  2. Define the Overview: In the Overview tab, click the title to rename your project. Click the description area to add details, goals, and next steps using Markdown.
  3. Track Features: Go to the Features tab to add and manage tasks, bugs, or enhancements. Set their status to keep track of your progress.
  4. Link Resources: In the Resources tab, click Add Resource to link to your existing Notes and Snippets, or to add external URLs relevant to your project.

Notes

Working with Notes

Notes are the core of your knowledge base. Use them for anything from meeting minutes to technical documentation.

Step-by-Step Instructions:

  1. Create a Note: Go to the Notes page and click New. You can choose to start with a blank note or select a pre-made template.
  2. Link Notes Together: While editing a note, click the Link Note button in the toolbar. A search box will appear. Find and select the note you want to link to, and a special clickable link will be inserted.
  3. Generate Flashcards: In the editor, structure your text in a question/answer format:
    • Q: What is the capital of the Philippines?
    • A: Manila
    Then, click the Create Flashcards button and choose which deck to add them to.

Graph View & Annotations

Visualizing Your Knowledge

The Graph View turns your notes into a visual, interactive web of ideas, and the Annotation Layer lets you build diagrams on top of it.

Step-by-Step Instructions:

  1. Filter Your View: On the Graph page, use the Tag Tree on the left to filter what notes appear. Checking a parent tag (e.g., `programming`) will also show notes with child tags (e.g., `programming/javascript`).
  2. Interact with Notes: Click on any note (node) in the graph to see a quick preview. If a node has a +/- icon, click it to collapse or expand its connected notes.
  3. Use the Annotation Layer: In a graph view tab, use the toolbar at the top-left to add diagrams. You can add draggable and resizable sticky notes, shapes, tables, and icons to create rich visual context. Click the "connector" button on an item to draw arrows between them.
