Welcome to Dev2ndBrain

This guide will walk you through setting up, using, and mastering the application to create your own powerful, interconnected knowledge base for all your development projects. Dev2ndBrain is a local-first application designed to be a developer's "second brain," ensuring your data is private, secure, and always available offline.

1. Installation & Initial Setup

Critical First Step: Configuring `appsettings.json`

To connect your GitHub account for Gist syncing and cloud backups, you must configure the application's backend settings. This is a one-time setup.

  1. Locate the Backend Folder: After installation, navigate to the following folder. Replace [User] with your Windows username.

    Path: C:\Users\[User]\AppData\Local\Programs\dev2ndbrain\resources\backend

  2. Create a GitHub OAuth App:
    • Log in to GitHub and go to Settings > Developer settings > OAuth Apps > New OAuth App.
    • Use the following configuration:
      • Application name: Dev2ndBrain Local
      • Homepage URL: http://localhost:5174
      • Authorization callback URL: http://localhost:5174/oauth/callback (This must be exact!)
    • Click Register application.
  3. Generate a Client Secret: On the next page, copy your **Client ID**. Then, click **Generate a new client secret** and copy the secret immediately.
  4. Edit the `appsettings.json` File: In the `backend` folder, open `appsettings.json` and replace its contents with the template below, pasting in your credentials. 
    {
  "Logging": { "LogLevel": { "Default": "Information", "Microsoft.AspNetCore": "Warning" } },
  "AllowedHosts": "*",
  "ConnectionStrings": { "DefaultConnection": "Data Source=dev2ndbrain.db" },
  "GitHub": {
    "ClientId": "PASTE_YOUR_CLIENT_ID_HERE",
    "ClientSecret": "PASTE_YOUR_CLIENT_SECRET_HERE"
  },
  "FrontendOrigin": "http://localhost:5174",
  "Gemini": { "ApiKey": "YOUR_GOOGLE_AI_STUDIO_API_KEY_HERE" }
}
  5. Save and Restart: Save the file and restart the Dev2ndBrain application. Your app is now fully configured!

Optional: Import Sample Data for a Quick Tour

Want to see how the app works without creating content from scratch? The installer includes a sample data file that will populate the application with pre-made notes, projects, snippets, and flashcards.

How to Import the Sample Data:

  1. Go to the Settings page using the left sidebar.
  2. Find the **Database Management** section.
  3. Click the **Import Data** button.
  4. A file picker will open. Navigate to the application's installation directory (usually `C:\Users\[User]\AppData\Local\Programs\dev2ndbrain`) and select the file named dev2ndbrain_db_local.json.
  5. Read the confirmation warning carefully. This will overwrite any data you've already created. Click "OK" to proceed.
  6. The application will process the file and then automatically reload, presenting you with the fully populated sample data. Now you can explore!

The Dev2ndBrain Interface

  • Sidebar: Your main navigation on the left. Access all major pages like Notes, Projects, and Settings.
  • Header: Contains the global search bar, GitHub login status, and the button to collapse/expand the sidebar.
  • Command Palette: Your power tool. Press Ctrl+K to open a search bar that can find anything and create new items on the fly.
  • Tab Bar: Located below the header, it allows you to switch between main pages and any items (notes, projects, etc.) you've opened in a new tab.
  • Main Content Area: The central part of the screen where you'll interact with your notes, projects, and other content.

A Deep Dive into Each Page

Dashboard

The Dashboard is your application's homepage and mission control center.

  • Flashcard Summary: Shows how many flashcards are currently due for review, encouraging you to keep up with your learning.
  • Recent Items: Provides quick links to your most recently edited notes and projects.
  • Quick Actions: Buttons to instantly create a new Note, Snippet, or Deck.

Notes

The Notes page is where you'll do most of your writing. It uses a powerful Markdown editor and allows you to create a web of interconnected knowledge.

Advanced Guide: Mastering Markdown

Markdown is a lightweight markup language for creating formatted text. Here are some essentials:

  • Headings: Start a line with # for a large heading, ## for a smaller one, and so on, up to ######.
  • Emphasis: Wrap text in *asterisks* for italics and **double asterisks** for bold.
  • Lists: Start a line with -, *, or 1. to create a list item.
  • Code: Use backticks for inline `code()`. For larger code blocks, wrap your code in triple backticks, specifying the language for syntax highlighting:
    ```javascript
    // your code here
    ```
  • Blockquotes: Start a line with > to create a quote.
  • Adding Images: To add an image from a URL, use the Markdown syntax: ![alt text](https://url.to/your/image.png). For more control over size, use an HTML <img> tag: <img src="https://url.to/image.png" width="300">

Linking Notes: The Core of Your Second Brain

To create a link to another note, type [[. A search box will appear, allowing you to select the note you want to link to. This creates a bi-directional link that powers the Graph View.

Using the Note AI Assistant

Click the "Ask AI" button in the editor toolbar for powerful actions:

  • Summarize: Condenses long notes into key bullet points.
  • Find Action Items: Extracts and elaborates on any to-do tasks or action items found in your text.
  • Explain This: Explains complex topics from your notes in simpler terms.

Snippets

A dedicated, searchable library for all your essential code snippets.

  • Running Code: `React`, `HTML`, `JavaScript`, and `C#` snippets have a "Run" or "Popout" button that executes the code in a sandboxed environment so you can test it instantly.
  • AI Assistant: Ask the AI to explain what a snippet does, suggest refactoring improvements, or ask a custom question about the code.
  • Gist Sync: After logging in with GitHub, you can sync any snippet to a private GitHub Gist, or push/pull updates between your local copy and the cloud.

Projects

The Project Workspace is the central hub for managing a project's lifecycle, from idea to completion.

  • Overview Tab: A showcase of the project's title, Markdown-enabled description, and prominent cards for the **Current Goal** and **Next Step**.
  • Features Tab: A searchable list to track all work items (features, bugs, etc.) with their status and description.
  • Resources Tab: A searchable table of linked materials. Notes and Snippets are **clickable links** that open in a new app tab.
  • History Tab: A complete, time-stamped audit trail with powerful search and filtering capabilities.

Flashcards

Master new topics with a powerful spaced repetition system (SRS).

  • Decks & Cards: Organize your knowledge into decks, and create cards with questions and answers.
  • SRS Review: When you review cards, you'll rate how well you knew the answer. The app's algorithm will then schedule the next review for that card at the optimal time to maximize retention.
  • AI Review: After a review session, you can ask the AI to analyze your performance, giving you feedback on your answers and suggestions for improvement.

Graph View

Visualize the connections between your notes to discover new insights.

  • Filtering: Use the tag tree on the left to filter the graph and focus on specific topics.
  • Annotation Layer: Use the toolbar in the graph to add shapes, text, icons, and arrows, turning your knowledge graph into an infinite mind-mapping canvas.

Settings

Database Management (Backup & Restore)

This is the most important feature for safeguarding your data. It allows you to create a complete, portable backup of your entire second brain.

Exporting Your Data
  1. Navigate to the **Settings** page.
  2. In the "Database Management" section, click the **Export Data** button.
  3. A JSON file named dev2ndbrain-backup-YYYY-MM-DD.json will be downloaded.
  4. Store this file in a safe, secure location (like a cloud drive or external hard drive).
Importing Your Data

⚠️ Warning: This Will Overwrite Everything!

Importing a file is a destructive action. It will **completely erase all current data** in the application and replace it with the data from the backup file. This cannot be undone.

  1. Navigate to the **Settings** page.
  2. Click the **Import Data** button. A file picker will open.
  3. Select the .json backup file you previously exported.
  4. Read the confirmation warning carefully and click "OK" to proceed.
  5. The application will process the file and then automatically reload, presenting your fully restored data.

Web Clipper Extension

The Dev2ndBrain Chrome extension allows you to clip content from any webpage directly into your notes.

  1. Install the extension from the Chrome Web Store (link to be provided).
  2. Select any text on a webpage.
  3. Right-click the selection and choose "Save to Dev2ndBrain".
  4. A new tab will open in your browser, automatically creating a new note with the clipped content, a link to the source page, and the tag #clipped.
